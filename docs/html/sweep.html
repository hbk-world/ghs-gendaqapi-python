GHS.ghs_cmd_trigger_arm() → str[source]

Arm the trigger, so that the next trigger will be accepted. * After the next trigger occurred, triggers are disarmed automatically and need to be * armed explicitly again using this function.

Read - This method can be called by multiple connected clients at same time.

Returns:

  • GHSReturnValue - API return values
